# 基于A\*与RRT算法的移动机器人路径规划复现

机器人控制技术课程大作业 | 张通 | 23电气 | 2308413009

## 项目简介

本项目复现了两种经典的移动机器人路径规划算法：

* **A\*算法**：基于栅格的启发式搜索算法，可保证路径最优性
* **RRT算法**：基于随机采样的快速搜索树算法，适用于高维空间

在两种不同复杂度的二维栅格地图环境下进行对比实验，评价指标包括路径长度、运行时间、节点扩展数量和路径平滑度。

## 环境要求

* Python 3.11+
* NumPy
* Matplotlib

## 数据说明

实验数据为程序自动生成的二维栅格地图：

* **简单地图**：50×50栅格，25%随机散点障碍物，模拟开阔场地
* **复杂地图**：50×50栅格，连续墙体结构+散点障碍物，模拟室内走廊环境
* 起点固定在左下角附近，终点固定在右上角附近
* 障碍物用1表示，自由空间用0表示

代码结构
path\_planning.py
├── 字体配置（自动检测系统字体，解决中文乱码）
├── generate\_map()          # 地图数据生成（简单/复杂两种模式）
├── AStarPlanner            # A\*算法实现（heapq优先队列，8方向移动）
├── RRTPlanner              # RRT算法实现（碰撞检测+路径重建）
├── calc\_path\_length()      # 路径长度评价
├── calc\_smoothness()       # 路径平滑度评价
└── main()                  # 主实验流程
